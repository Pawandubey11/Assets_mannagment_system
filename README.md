## 📋 About the Project

The Assets Management System is a comprehensive web application designed to help organizations efficiently track, manage, and maintain their assets. Built with a modern tech stack featuring React on the frontend and a robust Node.js backend, this project demonstrates enterprise-level application architecture with clean code, responsive design, and real-world business logic.

Whether you're managing IT equipment, office supplies, or company vehicles, this system provides the tools needed for seamless asset lifecycle management.

---

## ✨ Features

### Asset Management
- ✅ Create, read, update, and delete asset records
- ✅ Track asset ownership and assignment history
- ✅ Categorize assets by type, department, and location
- ✅ View active and inactive assets with filtering

### User Management
- ✅ Role-based access control (Admin/User)
- ✅ User authentication and authorization
- ✅ Department-wise asset assignment

### Dashboard & Analytics
- ✅ Interactive dashboard with asset overview
- ✅ Real-time asset status tracking
- ✅ Visual charts and statistics

### UI/UX
- ✅ Modern, responsive design with Tailwind CSS
- ✅ Smooth animations with Framer Motion
- ✅ Lucide React icons for a polished look
- ✅ Dark/Light mode support

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **npm** or **yarn**
- **Git**

### Installation

```bash
# Clone the repository
git clone https://github.com/Pawandubey11/Assets_mannagment_system.git

# Navigate to the project directory
cd Assets_mannagment_system

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:3000`

### Build for Production

```bash
# Create a production build
npm run build

# Preview the production build
npm run preview
```
